# JobTrack Pro 🚀

A full-stack job tracking application to manage job applications, interviews, and progress.

## 🛠️ Tech Stack

* Backend: Spring Boot
* Database: PostgreSQL (Neon)
* Frontend: React (coming soon)

## ✨ Features

* User management
* Job tracking (upcoming)
* REST APIs

## 🚀 Setup Instructions

### Backend

1. Clone repo
2. Configure DB in `application.properties`
3. Run Spring Boot application
